## Counsel : 심리상담사를 기준으로 한 상담자 관리 서비스

> 개발 기간 
- 2024.03.11 ~ 2024.05.17
<br>

> Stack
- **AI** : Python, Tensorflow 2.15.1
- **FrontEnd** :  HTML, CSS, JavaScript
- **BackEnd** : Flask

<br>

> 담당 업무
- Tensorflow를 활용하여 LSTM 기반 긍정 및 부정 감정 인식 모델 학습
- HTML, CSS를 활용한 웹페이지 구현
- Flask를 활용한 로컬 웹서버 구현
